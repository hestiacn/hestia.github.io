This website is built using Debian server operating system and Hestia server control panel.

1. Install the server operating system as Debian. 2. Install the Hestia server control panel.

Open the official website of Hestia Server Control Panel.

If you need a separate file management program for your website, we recommend using the File Manager used on this website.

3. By default, the Hestia server panel's files are in English after installation!
If you want to change it to Chinese, please complete steps 1 and 2 and then copy the corresponding server script code below and execute it on your server.

Debian server, copy the following code: (PS: Click the copy button to do so)

wget -c https://dns.Example.com/tihuan.sh && sh ./tihuan.sh

Ubuntu server, copy the following code: (PS: Click the copy button to do so)

wget -c https://dns.Example.com/utihuan.sh && sh ./utihuan.sh

Copy the script command above and execute it in your terminal. The script will automatically replace the relevant files. (PS: It will not cause any crashes! Please use it with confidence.)
